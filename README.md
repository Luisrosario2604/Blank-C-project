## Compiling

###### Go to the project directory

Clean (*.o files)
```
> make clean
```

Full clean (*.o and binary file)
```
> make fclean
```

Compile
```
> make
```

Full clean and compile
```
> make re
```

Compile with gdb (debugging)
```
> make gdb
```

Run tests
```
> make tests_run
```

Compile bonus
```
> make bonus
```
